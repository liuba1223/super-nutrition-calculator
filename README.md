# 薄荷健康营养查询 - Obsidian 插件

通过薄荷健康开放API查询食物和食材的营养成分信息，方便在Obsidian中记录和管理饮食营养数据。

## 功能特性

- **食物搜索**: 通过关键词搜索薄荷健康数据库中的食物
- **营养详情**: 查看食物的完整营养信息（热量、三大营养素、维生素、矿物质、氨基酸等）
- **健康提示**: 显示食物的健康灯标识和警告信息
- **快速插入**: 一键将营养信息以Markdown表格格式插入到当前笔记
- **选中查询**: 选中文字后直接查询该食物的营养信息

## 安装方法

### 手动安装
1. 下载本插件的 `main.js`、`manifest.json` 和 `styles.css` 文件
2. 在你的 Obsidian 库中创建文件夹: `.obsidian/plugins/boohee-nutrition/`
3. 将下载的文件复制到该文件夹
4. 重启 Obsidian
5. 在设置 -> 第三方插件中启用 "薄荷健康营养查询"

### 从源码构建
```bash
cd obsidian-boohee-nutrition
npm install
npm run build
```

## 配置说明

1. 前往 [薄荷健康开放平台](https://fc.boohee.com) 注册开发者账号
2. 创建应用获取 `App ID` 和 `App Key`
3. 在 Obsidian 设置 -> 薄荷健康营养查询 中填入你的凭证

## 使用方法

### 方式一：点击图标
点击左侧边栏的苹果图标 🍎 打开搜索框

### 方式二：命令面板
按 `Ctrl/Cmd + P` 打开命令面板，搜索：
- "搜索食物营养信息" - 打开搜索框
- "查询选中文字的营养信息" - 查询当前选中的文字

### 方式三：选中查询
1. 在笔记中选中食物名称（如"苹果"）
2. 使用命令面板执行"查询选中文字的营养信息"

## 插入示例

查询"苹果"后插入到笔记的效果：

```markdown
## 苹果

> ✅ 低热量

### 热量信息
| 项目 | 数值 |
|---|---|
| 总热量 | 54Kcal |

### 三大营养素
| 营养素 | 含量 |
|---|---|
| 碳水化合物 | 13.5g |
| └ 糖 | 10.1g |
| └ 膳食纤维 | 1.2g |
| 蛋白质 | 0.2g |
| 脂肪 | 0.2g |
```

## API 说明

本插件使用薄荷健康食物数据开放平台 API：
- 域名: `https://fc.boohee.com`
- 文档: https://fc.boohee.com/panel/open_api_docs/document
